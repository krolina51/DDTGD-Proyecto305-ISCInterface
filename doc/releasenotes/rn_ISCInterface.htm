ISCInterface v2.6

Autor: Javier Flores
Fecha: 2021.05.03
Entidad: Banco de Bogotá
Instalador: ISCInterface_v2.6_build2105031_r5.6.exe
Descripción Funcional: Ajustes de Next Day y transacción de deposito.
Descripción Técnica: Se modifica la persistencia del estado del parametro next day para las interchanges configuradas una vez se reinicie la interfaz,se mantenga el estado. Para la transacción de deposito se eliminan los caracteres no imprimibles recibidos desde el autorizador pro espacios.

ISCInterface v2.9

Autor: Javier Flores
Fecha: 2021.07.21
Entidad: Banco de Bogotá
Instalador: ISCInterface_v2.9_build2107211_r5.6.exe
Descripción Funcional: Ajustes mesajes 0220 y flag Next Day.
Descripción Técnica: Se ajusta el envió de mensajes 0220 al autorizador, para que solo se envien si vienen con el tag transaction_to_host y viene el con el valor Mandatory. DE lo contrario el mensaje se respondera automáticamente en la interfaz. Se reemplaza el archivo NextDay.properties con el nuevo formato donde el parametro a leer y modificar es NEXTDAY.
