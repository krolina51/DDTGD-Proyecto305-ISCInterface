Data Security Standards

The PCI Security Standards Council (PCI SSC) has defined a set of best practices, called the Payment Application Data Security Standard (PA-DSS), which are aimed at assisting software vendors to create secure payment applications. For a complete listing of S1 Corporation components and versions that are PA-DSS validated, please see the PCI website: https://www.pcisecuritystandards.org/

The following statement from the Payment Card Industry PA-DSS Vendor Release Agreement between the PCI SSC and S1 Corporation applies only to those licensed products listed on the PCI website as being approved:
