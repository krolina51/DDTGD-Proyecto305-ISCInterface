Instrucciones de Instalación

  1. La interface DvcBogota es instalada ejecutando el programa Setup.exe en un Postilion Server.

  2. Seleccione "Server Installation" cuando se presente en el dialogo "Client or Server Installation" .

  3. Durante la instalación, seleccione una cuenta con derechos administractivos en Postilion Server. Esto por lo general la cuenta administrador

  4. Configure la interface Postilion DvcBogota en el sistema Postilion. Ver Descripción General - Configuración.

  5. Para ejecutar el archivo que poblará la base de datos para esta interface, desde la línea de comando, ejecutar el archivo batch localizado en la carpeta "bin" en el folder con los scripts para esta interface. Si encuentra problemas ejecutando este archivo desde la línea de comando, por favor dirigirse a su proveedor de Postilion o al correo postilion.support@processa.com
