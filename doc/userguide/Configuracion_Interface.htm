Parametro

Descripción

SAP

Crear un Service Access Point (SAP) en la categoria SAPs. defina el protocolo de comunicación a ser usado por el servidor :

Variable

Setting

SAP

Cualquier nombre. Se sugiere que este nombre se construye mediante la adición del sufijo "Sap" para el intercambio nombre.

Si más de un SAP se define para una interchange, se sugiere que una nueva se añade el sufijo numérico. Por ejemplo, llame a los dos programas de ajuste estructural para un intercambio MyInterchange, MyInterchangeSap1 y MyInterchangeSap2 respectivamente.

Type

Seleccione client. Un client SAP intentos de estblecer comunicacion con la entidad remota.

Protocol

Seleccione Generic protocol.

SAP Factory class name

postilion.DvcBogota.ipc.TcpSapFactoryCustom

Address

Ninguna. Crear una conexión en la sección de Conexiones y proveer un puerto y máquina "localhost" para comunicarse con la entidad remota
