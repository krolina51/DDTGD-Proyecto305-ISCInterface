Descripción General

La interfaz DvcBogota actuará como puente entre Postilion y la entidad autorizadora (en este caso Banco de Bogotá). Todos los mensajes enviados desde y hacia Postilion se adhieren a la implementación del estándar Iso8583 hecha por el kit de desarrollo de Postilion. Los mensajes desde y hacia la entidad remota se adhieren al estándar de Base-24.

La interfaz Postilion DvcBogota soporta las siguientes comandos de monitoreo:

  • Open/Close interchange

  • Enable/disable tracing

 

Para mas información referirse a Instructiones Control / Monitor.
