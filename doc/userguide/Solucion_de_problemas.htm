Solución de problemas

Esta interfaz de postilion registra errores, eventos de información y alertas cuando se producen determinadas condiciones. Estos errores se registran en el visor de eventos de Windows

En esta sección se enumeran todos los acontecimientos que esta interfaz de Postilion es capaz de processar. cada evento proporciona lo siguiente:

  • Número de eventos

  • Severidad

  • Nombre simbolico

  • Mensajes de texto y descripción de parameter

  • Descripción

  • Acción
