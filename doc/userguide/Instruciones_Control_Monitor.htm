Instruciones de Control / Monitor

La interface Postilion DvcBogota responde a los siguientes comandos usando Postilion Monitor como se especifica a continuación:

TRACE

  • TRACE <Nombre de la Interchange> <Parámetro>

<Nombre de la Interchange>: El nombre asignado a la Interchange

<Parámetro>: ON/OFF/CLEAR/?

Si el parámetro es ON, el trace está encendido. Si el parámetro es OFF, el trace se deshabilita o apaga. Si el parámetro es CLEAR los traces que existan serán borrados. Si el parámetro es ? la interchange retorna si el trace está encendido o apagado.

Si el trace está encendido o habilitado, el archivo de trace muestra los mensajes de <Postilion>\Trace\AthNode\<Interchange>. Los archivos del rastro están en formato del HTML y se pueden ver con un browser del Internet tal como Microsoft Internet Explorer.

 

OPEN

 

  • OPEN <Nombre de la Interchange>

<Nombre de la Interchange>: El nombre asignado a la Interchange

El comando OPEN solicita establecer una conexión de Red y una petición sing-on. Este comando es utilizado típicamente después de haber utilizado un comando CLOSE, por lo tanto un comando OPEN será ignorado a menos que se haya ejecutado un comando CLOSE.

 

CLOSE

  • CLOSE <Nombre de la Interchange>

<Nombre de la Interchange>: El nombre asignado a la Interchange

El comando CLOSE realiza una petición sing-off y cuando recibe la respuesta cierra la conexión de red. Un comando CLOSE no será ejecutado a menos que exista una conexión con la Interchange.
