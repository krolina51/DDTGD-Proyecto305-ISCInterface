Configuración Nodo Sink

Por la consola de Transaction Manager Configurator en la opción de Sink Node se deben configurar los siguientes parámetros.

Variable

Escenario

Node

Cualquier nombre. Se recomienda que los 3 primeros caracteres del nombre correspondan al canal de la red. La longitud maxima de este campo es 12 caracteres. El nombre del nodo debe ser unico.

Providing service to

Seleccione "Acquirer"

Authorization Service

Seleccione "none"

IP Address

la dirección IP (o nombre del host) de la maquina en la cual este corriendo la Interface. Si la interfaz esta corriendo en la misma maquina de Transaction Manager, este valor debe ser localhost.

Port Number

El puerto TCP de la interfaz DvcBogota en el cual va a escuchar para una conexion de Transaction Manager.

Completion Indicator

Negative

Retention Period

Cualquier valor. Se recomienda en sistemas en producción que este valor no sea mayor a 15

Timeout Period

Tipicamente se colocan 30 segundos. El tiempo de espera de las notificaciones en el nodo sink debe ser menor al tiempo de espera de las notificaciones enviadas por las entidades remotas.

Advice Tmeout Period

Valor por defecto

Advice declined and zero value transactions

No se notificarán transacciones declinadas o de valor 0. Esta opción no es seleccionada.

Advice Inquiry Transactions

No se notificarán las transacciones de consulta. Esta opción no es seleccionada.

Allocate System Trace Audit Number

No se manejará el número de auditoría de las transacciones en el Transaction Manager. Esta opción no es seleccionada.

Flush Store-and-Forwartd

Las transacciones de requerimiento tendrán prioridad sobre las notificaciones. Esta opción no es seleccionada.

 

Decline Hotcard

Seleccionar esta opción

Decline invalid address

Seleccionar esta opción

Route to Source

No seleccionar esta opción

Do not reverse in-flight transactions

Seleccionar esta opción (No se reversarán transacciones en vuelo).

Respond to duplicate transactions

Seleccionar esta opción (Se responderán a transacciones duplicadas).

Notify Acquirer File Update Advices

No seleccionar esta opción

Active/Active

Seleccionar esta opción (Se manejará el esquema Activo/Activo)

Early Notification for advices

No seleccionar esta opción

Batch Management

Seleccionar "Use External Business Date". La fecha contable utilizada será aquella que venga del nodo source correspondiente

PIN Key

El nombre de la Issuer Working Key (o KWP) para la Interchange en caso que la interfaz . Ver Configuración Criptografica.

Batch Processing Deadline

23:00

External Cutover Prev Date

No seleccionar

Reconcile per

Node

Currency

Colombian Peso

Notify PIN failure?

No seleccionar

Translate PIN?

Seleccionar. El nodo sink espera el PIN block cifrado con la misma llave que posee la entidad remota.
